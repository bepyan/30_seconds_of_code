# 30_seconds_of_code
Study code from https://www.30secondsofcode.org

### How to see preview ?
1. clone project
    ```shell
    cd [your directory]
    git clone https://github.com/bepyan/30_seconds_of_code
    cd 30_seconds_of_code
    code .
    ```
2. use vscode `Live Server` plugin
